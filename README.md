# 📚 Maktebti – Online Library Platform

## 📌 Description
Maktebti is a front-end web application designed to simulate an online library experience. The platform allows users to explore library services such as borrowing, searching, reserving books, and accessing e-reading features through an intuitive and modern interface.

The goal of this project is to create a clean and user-friendly UI while applying fundamental UI/UX design principles.

---

## 🎯 Objectives
- Design a simple and intuitive library interface
- Improve user navigation and interaction
- Apply UI/UX design concepts in a real project
- Practice front-end development using HTML and CSS

---

## 🚀 Features

### 🔐 Authentication System (UI)
- Sign In modal
- Sign Up modal
- My Account interface with user information and statistics

### 📚 Library Services
- Borrow books via a modal form
- Search books by name
- Reserve books for later
- Access e-reading functionality

### 🖥️ User Interface
- Responsive homepage with introduction
- Clean navigation header
- Service cards with interactive modals
- Contact form section

---

## 🛠️ Technologies Used
- HTML5
- CSS3
- Google Fonts (Poppins)

---

## 🎨 UI/UX Design Focus
This project emphasizes:
- Clean and structured layout
- Consistent typography and spacing
- User-friendly navigation
- Interactive components (modals)
- Visual hierarchy for better readability
